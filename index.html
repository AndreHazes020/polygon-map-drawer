Enable Location

To use your current location, please enable location access in your browser settings.

iPhone / iPad

Settings → Safari → Location → Allow

Android

Tap the lock icon in browser → Permissions → Location → Allow

Desktop

Click the lock/info icon in the address bar → Location → Allow

How to Use

Search

Type any location to fly there instantly.

Draw

Tap points on the map. Double-tap to finish.

Copy

Copies your polygons as GeoJSON to clipboard.

Locate

Jump to your current location.
