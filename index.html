Enable Location

Location access is blocked. You can either enable it in your browser settings, or use the Search bar to find your restoration site.

Alternative: Close this and use the search bar at the top to find your location by name.

To enable location access:

iPhone / iPad

Settings → Safari → Location → Allow

Android

Tap the lock icon in browser → Permissions → Location → Allow

Desktop

Click the lock/info icon in the address bar → Location → Allow

How to Use

1

Find Your Location

Tap Locate to jump to your current position, or use the Search bar to find your restoration site.

2

Draw the Restoration Area

Tap Draw, then tap points around your restoration area. Connect the last point to the first to close the shape.

3

Finish & Return

Tap Finish to copy the polygon data and return to the form where you can paste it.

Made a mistake? Tap Clear to remove all drawings and start over.
